Final data analysis project for Udacity's Data Analysis Term 1 - Nanodegree

Question: What features are most important for predicting whether a patient will show up to their scheduled appointment or not? What accuracy if any can be attributed to each analysis method, namely feature importance and Chi-squared?  Additionally, what day of the week is related to the highest proportion of 'No-show's?

This report includes exploratory data analysis, data wrangling, and statistical modelling.
